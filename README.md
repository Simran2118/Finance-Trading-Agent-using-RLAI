# Finance Trading Agent

## Overview

Finance Trading Agent is a Reinforcement Learning-based stock trading system developed using historical stock market data from Tata Motors and Paytm. The project automates trading decisions by training an RL agent in a custom trading environment and evaluates its performance through backtesting and visualization.

## Features

* Historical stock data collection using Yahoo Finance
* Data preprocessing and feature engineering
* Technical indicators including RSI, MACD, Moving Averages, and Bollinger Bands
* Custom trading environment built using Gymnasium
* Reinforcement Learning agent training using Stable-Baselines3
* Portfolio performance evaluation and visualization
* Automated Buy, Sell, and Hold decision-making

## Technologies Used

* Python
* Reinforcement Learning
* Gymnasium
* Stable-Baselines3
* Pandas
* NumPy
* Matplotlib
* yFinance
* Technical Analysis (TA) Library

## Project Workflow

1. Collect historical stock market data.
2. Preprocess data and generate technical indicators.
3. Create a custom stock trading environment.
4. Train the Reinforcement Learning agent.
5. Evaluate trading performance and portfolio returns.
6. Visualize results and analyze trading metrics.

## Results

The agent learns trading strategies from historical market data and makes trading decisions to maximize portfolio performance while adapting to changing market conditions.
